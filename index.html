Project Background & Solution

1. 추진 배경 (Why Now?)

  • 일본의 야생 곰 피해 급증: 도심 출몰 빈도 증가로 인한 인명 및 재산 피해 심각화
  • 국내 잠재적 위협 증가: 반달가슴곰 개체 수 증가에 따른 민가/등산로 출몰 가능성 대비 필요

2. 기존 감시 체계의 문제점 (Pain Points)

  • 중앙 서버 집중형의 한계: 광대한 산간 지역의 모든 영상을 메인 서버로 전송하여 탐지하는 것은 불가능 (네트워크 및 비용 제약)
  • 실시간 탐지 불안정: 서버 과부하 시 실시간 탐지 실패 우려

3. 핵심 솔루션 (Our Solution)

  • On-Device AI (Edge Computing): NPU가 탑재된 엣지 단말(CCTV 등) 활용
  • YOLO Object Detection: 경량화된 객체 탐지 모델을 탑재하여 현장에서 즉시 분석

4. 기대 효과 (Expected Effects)

  • 효율적 감시 체계 구축: 소규모 장비로도 광범위한 지역의 곰 출몰을 실시간으로 감지 가능
